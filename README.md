# Jasmine Flora J - Personal Portfolio

This is a static single-page personal portfolio website built with HTML5 and CSS3.

## Features
- Pure CSS Image Slider
- Responsive Design (Mobile First)
- PWA Ready (Manifest included)
- Contact Form (UI only)
- Skills Progress Bars (CSS)

## Setup
1. Unzip the project.
2. Open `index.html` in any modern web browser.

## Technologies
- HTML5
- CSS3
- No JavaScript
- No External Libraries

## Project Structure
- `index.html`: Main portfolio page.
- `css/style.css`: All styles.
- `assets/`: Images and icons.
